# Keewerd

Keewerd is a lightweight web app for creators that helps turn trend signals into ready-to-post publishing copy. It is designed for video editors, musicians, photographers, designers, and streamers who repeatedly need better titles, descriptions, tags, hashtags, and posting guidance.

The goal of Keewerd is simple: given a platform, content type, topic, and style, help the user figure out what to title their content, how to describe it, which tags to use, and when to publish it.

## Stack

- TypeScript
- Tailwind CSS
- Vercel

## Product Positioning

Keewerd is a **trend-informed publishing assistant for creators**.

It is **not**:
- a full social media management suite
- a guarantee engine for virality
- a fake all-platform live analytics dashboard

## Initial MVP Goals

- simple no-login experience
- dropdown-heavy workflow
- copy-paste-ready outputs
- title, description, tag, and timing guidance
- source-backed trend summaries

## Planned Routes

- / — landing page

- /about
- /privacy
- /terms

## Development Notes

The MVP will prioritize simplicity, legal/platform-safe data sourcing, and fast deployment over feature bloat. The first launch will focus on useful outputs and measurable traffic rather than accounts, subscriptions, or complex dashboards.
